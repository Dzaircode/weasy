Add Poppins font files here so the app can use the Poppins font:

1. Download Poppins from Google Fonts: https://fonts.google.com/specimen/Poppins
2. Click "Download family" and unzip.
3. Copy these two files into this folder (assets/fonts/poppins/):
   - Poppins-Regular.ttf
   - Poppins-Bold.ttf

Then run: flutter pub get
